# Naruto Hand Gesture Project

**GitHub Repository:** https://github.com/Sevo1730/Naruto1  
**Live Demo:** https://sevo1730.github.io/Naruto1/

---

## Ашигласан технологиуд

- **HTML5 / CSS3 / JavaScript** — вэб хуудасны үндсэн бүтэц
- **MediaPipe Hands** — гарын хөдөлгөөнийг бодит цаг хугацаанд таних AI сан
- **MediaPipe Camera Utils** — камераас зураг авах хэрэгсэл
- **Canvas API** — гарын ясны дүрслэлийг зурах
- **HTML5 Video** — Naruto болон Sasuke-ийн эффектийн видео тоглуулах

---

## Төслийн товч тайлбар

Энэ бол вэб камераар хэрэглэгчийн гарыг бодит цаг хугацаанд таньж, гарыг нээх үед Naruto болон Sasuke-ийн тусгай эффектийг дэлгэц дээр харуулдаг интерактив вэб апп юм. Зүүн гараар Naruto, баруун гараар Sasuke-ийн эффект гардаг.

---

## Төслийн үндсэн зорилго

Хиймэл оюун ухааны гарын таних технологийг ашиглан хөгжилтэй, интерактив вэб апп хийх. Кодчиллын мэдлэгээ практикт хэрэглэж, MediaPipe зэрэг бодит AI хэрэгсэлтэй ажиллах туршлага хуримтлуулах нь гол зорилго байсан.

---

## Тулгарсан бэрхшээлүүд

- MediaPipe-ийн WASM файлуудыг зөв ачаалах — CDN холбоос тохируулах шаардлагатай байсан
- Хоёр гарыг зэрэг таних үед зүүн/баруун гарыг зөв ялгах
- Гарын байрлалыг Canvas координат руу зөв хөрвүүлж эффектийг гарын дотор тавих
- GitHub Pages дээр видео файлуудыг зөв serve хийлгэх

---

## Бэрхшээлийг хэрхэн даван туулсан

- MediaPipe-ийн `locateFile` функцийг ашиглан WASM файлуудыг CDN-ээс татах замаар шийдсэн
- `multiHandedness` массивын `label` утгыг (`'Left'` / `'Right'`) ашиглан гар ялгасан
- Wrist болон Knuckle цэгүүдийн координатаар эффектийн байрлалыг тооцоолсон
- Видео файлуудыг репозиторид шууд оруулж, харьцангуй замаар (`assets/naruto.mp4`) дуудсан

---

## Төслөөс юу сурсан

- MediaPipe зэрэг бодит ертөнцийн AI сантай ажиллах арга
- Canvas дээр бодит цаг хугацааны зураг дүрслэл хийх
- Камер, видео, Canvas зэрэг олон HTML5 API-г хамт ашиглах
- GitHub Pages ашиглан статик вэб апп deploy хийх
